What is CSS?

  1. CSS stands for Cascading Style Sheets.

  2. CSS describes how HTML elements are to be displayed on screen,paper, or in other media.

  3. Css is not neccesary CSS saves a lot of work. It can control the layout web pages all at once

  4. External stylesheets are stored in CSS files.

CSS csn be added to HTML documents in 3 ways:

  • Inline - by using the style attribute inside HTML elements.
  • Internal - by using a style element int the head section.
  • External - by using a link element to link to an external CSS file.
  • The most common way to add CSS , is to keep the styles in external CSS file. However, in this tutorial we will use inline and internal styles, because this easier to demonstrate, and easier for you to try it yourself.
